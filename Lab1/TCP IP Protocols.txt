Протоколи TCP/IP: Протоколи TCP/IP - це конкретні комунікаційні протоколи, які входять у склад TCP/IP-архітектури і використовуються для реалізації комунікації в комп'ютерних мережах. Основні протоколи TCP/IP включають в себе:

IP (Internet Protocol): Відповідає за призначення та маршрутизацію пакетів даних в мережі. IP надає адреси пристроям та визначає, як пакети повинні бути доставлені.

TCP (Transmission Control Protocol): Відповідає за надійний обмін даними між пристроями. TCP гарантує, що дані будуть доставлені у правильному порядку та без втрат.

UDP (User Datagram Protocol): Використовується для передачі даних, але не гарантує надійність, що робить його підходящим для додатків, де швидкість важливіша за надійність.

ICMP (Internet Control Message Protocol): Використовується для обміну службовими повідомленнями між пристроями в мережі.
